🖥️ Comment Monter et Démonter son Ordinateur

Une initiation simple pour les débutants — avec précision et style

💬 “L’éducation est l’arme la plus puissante qu’on puisse utiliser pour changer le monde.” — Nelson Mandela

🧩 1. Préparation

  • 🔌 Éteindre et débrancher l’ordinateur
  • ⚡ Se décharger de l’électricité statique
  • 🧰 Rassembler les outils nécessaires
  • 🗂️ Organiser son espace de travail

🔧 2. Démontage des composants

Composant Étapes clés
💼 Boîtier Retirer les vis à l’arrière, ouvrir le capot
🧠 RAM Déclipser doucement les barrettes
💽 Disque dur / SSD Déconnecter câbles SATA/NVMe, retirer les vis
🎮 Carte graphique Déverrouiller le loquet, retirer délicatement
⚙️ Alimentation Débrancher les câbles, retirer les vis

🛠️ 3. Montage

  • 📍 Positionner et visser la carte mère
  • 🧬 Installer le CPU + ventirad avec pâte thermique
  • 🧩 Ajouter les barrettes de RAM
  • 📦 Fixer le disque dur / SSD et reconnecter les câbles
  • 🎨 Installer la carte graphique dans le slot PCIe
  • 🔌 Connecter l’alimentation et les câbles

✅ 4. Vérification finale

  • 🌐 Rebrancher les périphériques
  • 🚀 Allumer et vérifier les signaux
  • 🧪 Tester les branchements si problème
