🔰 Débuter en informatique

Bienvenue sur Afro Code TV ! Ce guide est conçu pour celles et ceux qui souhaitent comprendre les bases de l’informatique de manière simple et visuelle.

🧠 Le processeur

Le processeur est le “cerveau” de l’ordinateur. Il traite les informations et exécute les instructions que tu donnes à ton système.

💾 La mémoire vive (RAM)

Elle permet à l’ordinateur de travailler rapidement en stockant temporairement les données pendant son utilisation.

🗃️ Le disque dur

Le disque dur garde les fichiers, les documents et les programmes, même quand l’ordinateur est éteint.

🌐 Naviguer sur Internet

Internet est une immense bibliothèque mondiale. On y accède via des navigateurs (Chrome, Firefox, etc.), mais il faut rester vigilant pour sa sécurité.

⌨️ Premiers pas en codage (HTML)

Le HTML est la base des pages web. Voici un exemple : 

  <h1>Bienvenue</h1>
  <p>Ceci est un paragraphe.</p>
